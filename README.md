# DevBrief Generator

## Objectif Principal

Valider la capacité de l'IA à générer des briefs de projets de haute qualité, réalistes et pertinents pour les développeurs juniors, et prouver l'intérêt des développeurs pour ce type d'outil.

## Fonctionnalités Clés (MVP)

- Génération de Briefs : Interface simple pour saisir les préférences (domaine, stack, niveau, durée) et générer un brief détaillé.
- Base de Connaissances Curatée : Le cœur du système, alimentant l'IA avec des cas d'étude réels simplifiés et des tendances du marché.
- Format de Sortie : Briefs structurés, téléchargeables (Markdown/PDF).
- Gestion Utilisateur Basique : Création de compte, historique des briefs générés.
- Public Cible : Développeurs juniors (B2C).

## Valeur pour le Client

- Accès rapide à des idées de projets pertinents pour leur portfolio.
- Gain de temps dans la recherche de projets.
- Aide à la structuration de leur apprentissage.
